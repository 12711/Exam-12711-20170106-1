#jetty命令 mvn -Djetty.port=8888 jetty:run
浏览器输入后进去登录页面,如登录失败弹出对话框 提示登录失败如登录成功则跳转到电影展示页面(list.html)


list.html:功能
     1.分页显示
     2.可以通过模糊查询电影名进行电影的分页查询
     3.可以点击上一页下一页
     4.可用通过输入页码跳转到对应页面
     5.点击新增按钮调到新增页面(insert.html)
     6.删除一天记录
     7.修改一条记录(弹出框)
     8.可以指定页面显示的数据条数
